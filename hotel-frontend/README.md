# Hotel Booking Frontend

Frontend for our CSE 412 hotel booking project. Built with React.

## How to Run

1. Install dependencies:
```bash
npm install
```

2. Make sure the backend server is running on `http://localhost:3001`.

3. Start the app:
```bash
npm start
```

Opens at `http://localhost:3000`.

## Pages

- **Home** — landing page with navigation
- **Room Reservations** — book and manage room reservations
- **Space Reservations** — reserve event spaces
- **Rooms** — view all rooms
- **Guests** — manage guests
- **Upgrades** — view available upgrades
- **Staff** — staff info
